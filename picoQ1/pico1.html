Challenge Description

This challenge involves exploiting a simple SQL Injection vulnerability in a login form.

Writeup Steps

  1. Analyzed the login form and noticed user inputs are not sanitized.
  2. Injected `' OR '1'='1` to bypass authentication.
  3. Accessed the admin panel and retrieved the flag.
